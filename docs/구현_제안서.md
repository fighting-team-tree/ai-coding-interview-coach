## 2. 기술 명세

| 2.1 기술목적 | 기존 코딩 테스트 플랫폼은 논리적 사고력을 '설명'하는 면접 훈련을 제공하지 못함. 본 기술은 학습자가 작성한 코드를 기반으로 AI가 꼬리질문(Socratic Method)을 4~5 깊이로 던지는 과정을 통해, 개발자의 의사소통 능력, 코드 최적화 센스, 트레이드오프 분석 역량을 향상시키고 부트캠프 강사들의 기술 면접 개인지도 리소스를 획기적으로 줄이는 데 목적이 있음. |
| --- | --- |
| 2.2 기술구조 | **1. 입력 파이프라인:** 제출된 코드의 정적 분석(AST) 및 RAG JSON(문제 팩트, 취약점 함정 데이터) 동적 로딩<br>**2. 다중 에이전트 레이어 (LangGraph 활용):**<br>- Fact Checker(나침반): JSON 기준 절대 규칙을 LLM 컨텍스트 공간에 강제 주입<br>- Interviewer(면접관): 주입된 팩트의 바운더리 안에서 압박 질문을 유동적으로 생성<br>**3. 평가 레이어 (Tiered 추론 시스템):** 3~4턴의 텍스트 문답(저비용 모델 처리) 종료 후, 전체 로그를 취합하여 고성능 모델 기반 피드백 추론<br><br>**[구조도 요약]**<br>사용자 제출 코드 -> AST / RAG Fact 주입 -> LangGraph State Machine 기반 티키타카(3~4턴) -> 3축 종합 평가 프롬프팅 -> 리포트 UI 생성 |
| 2.3 주요기능 | - 파이썬 기반 알고리즘 문제 해결 및 코드 제출/실행 에디터<br>- 제출된 코드상의 취약점 기반 동적 꼬리 질문 및 스케일업(Scale-up) 압박 질문(Plan B) 지원<br>- 면접 종료 후 논리 구조/기술 정확도/설명 명확도 기반 3축 평가 리포트 발급<br>**- (신뢰도 확보) B2B 도입 강사가 AI의 최종 평가를 보정할 수 있는 휴먼 오버라이드(Human-Override) 대시보드**<br>**- (확장성 확보) 강사 전용 커스텀 문항 RAG JSON 자동 생성(Auto-generation) 저작 도구**<br>- 면접 종료 후 '논리적 사고력 성장 그래프' 중심의 SNS 공유용 피드백 리포트 카드 (바이럴 마케팅 장치) |
| 2.4 결과물 형상 | 웹 기반 SaaS 플랫폼 (채용 기관, 교육 기관 타겟 B2B 애플리케이션 형태) |
| 2.5 배포방식 | 반응형 웹 애플리케이션 기반 배포 (프론트/백엔드/데이터 스토어 연동 클라우드 인프라 활용) |
| 2.6 혁신적 요소 | - 프롬프트 자유도를 제약하고 '조종간(Fact)' JSON을 파라미터화하여 주입시킴으로써 LLM 특유의 할루시네이션(잘못된 지식 창작)을 강력하게 통제.<br>- **Optimistic UI 기반 제로 레이턴시(Zero Latency) UX:** 코드 제출 직후 채점 결과를 먼저 띄우는 수 초 동안, 백그라운드 비동기로 AST 분석과 면접관 인스턴스를 예열하여 유저의 '로딩 이탈'을 방지하고 완벽한 몰입 지원. |
| 2.7 도전적 요소 | 비정상적 스파게티 코드 입력 시 AI가 예측 맵핑에 실패할 수 있으므로 발동하는 **'동적 폴백(Fallback) 핵심 개념 질문'**, 반대로 약점이 없는 완벽한 복붙 코드 입력 시 발동하는 **'시스템 스케일업 압박 질문(Plan B)'** 등 양극단의 엣지 케이스를 LangGraph 상태 제어로 끊어짐 없이 처리하는 고난도 아키텍처. |

## 3. 구현 방법 및 계획

| 3.1 구현범위 | **세부업무 #1: RAG 나침반(Fact/Trap JSON) 데이터 파이프라인 및 코어 문항(5패턴) 셋업**<br>**세부업무 #2: 코드 파싱 및 LangGraph 기반 텍스트 면접관 에이전트(PoC) 코어 로직 구현**<br>**세부업무 #3: 심의 모델(고비용 모델) 평가 프롬프트 엔지니어링 및 3축 피드백 리포트 UI/UX 개발** |
| --- | --- |
| 3.2 구현계획 | - **1주차:** 핵심 알고리즘 5문항 Fact/Trap JSON 데이터 구축, AST 파서 프로토타입, 단일 턴 챗봇 PoC 검증<br>- **2주차:** LangGraph 기반 다중 에이전트 상태 머신 코어 로직 구현, 멀티턴 대화 플로우 검증<br>- **3주차:** Monaco Editor 기반 웹 에디터 통합, 챗봇 UI 구현, 3축 평가 리포트 프롬프트 엔지니어링<br>- **4주차:** 블라인드 테스트 20세트 검증, Optimistic UI 적용 및 UX 최적화, 피드백 반영 및 데모 비디오 산출 |
| 3.3 기술스택 | - **프론트엔드:** React.js, TailwindCSS (인지 부하를 낮추는 미니멀 텍스트 몰입형 UI)<br>- **백엔드/에이전트 로직:** Python (FastAPI), LangChain/LangGraph<br>- **데이터/인프라:** PostgreSQL 혹은 경량 Document DB, AWS 혹은 Vercel 기반 배포<br>- **핵심 AI 추론:** 경량 실시간 모델 (Haiku급/GPT-4o mini급 최신 모델, 실시간 면접), 고성능 추론 모델 (Sonnet급 최신 모델, 최종 평가 1회 호출) |
| 3.4 시설·장비 보유현황 | - **개발 장비:** Windows/macOS 개발 워크스테이션 (팀원 개인 보유)<br>- **클라우드 인프라:** AWS/Vercel 기반 배포 환경, LLM API 크레딧 확보 (Anthropic/OpenAI)<br>- **데이터셋:** Python 알고리즘 풀이 코드 100건 이상, 핵심 5패턴별 Fact/Trap JSON 자체 구축 예정 |

## 4. 파급효과

| 4.1 기술적 파급효과 | 기존 코딩 교육이 문제 해결 결과(Test cases passed)에만 집착하던 '정적 테스트 중심 패러다임'에서 나아가, 코딩의 의도, 복잡도 판단, 코드 컨텍스트 설명 능력을 묻는 '과정 지향적 Socratic 평가 패러다임'으로 AI 에듀테크 기술을 진일보시킵니다. 시스템에 채택된 제어형(Fact 강제 주입) 구조 모델은 추후 금융, HR 등 다른 B2B 도메인의 상담 훈련 AI 시스템에도 확장이 용이합니다. |
| --- | --- |
| 4.2 사회·산업적 파급효과 | - **비용(산업/확장) 관점:** 부트캠프 등의 B2B 심층 면접 인건비를 획기적으로 절감합니다 (강사 1인 기술 면접 시급 약 5만 원 × 수강생 100명 기준 월 500만 원 → AI 에이전트 API 비용 건당 약 500원 이내로 대체 가능). 특히 타 플랫폼 데이터 크롤링에 의존하지 않고, 강사가 직접 입력한 문항으로 AI가 면접 함정(Traps)을 자동 추출하는 **'자체 폐쇄형 생태계(Data Flywheel)'**를 구축하여 합법적이고 폭발적인 Scale-up이 가능합니다.<br>- **사회 관점:** 글로벌 IT 기업들이 요구하는 '지식을 설명할 줄 아는 소프트웨어 엔지니어링 역량'의 진입 장벽을 대폭 낮춰 교육 정보 격차를 해소합니다. |
